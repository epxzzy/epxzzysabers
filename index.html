epxzzy's sabers

High-speed duels. Stances. Abilities. Infinite colors.

About

epxzzy's sabers adds in several new weapons, some with unique gimmicks. These weapons have:

  • 8 unique weapon variants with more to come,
  • 5 abilities across 3 variants,
  • True combat: sabers function like shields when you block, and like axes when you strike
  • God so many colors: customisability down to individual RGB values
  • Deflection: ability to just send back incomming projectiles
  • Togglable states: ignite only when you want to
  • Stances: hold Shift + RMB to assume one of 8 stances
  • True animations: swings, strikes, blocks, stances, abilities, and exposed hitpoints, all visible in real time
  • …and so much more

FAQ

Can i put this in my modpack?

Yeah, as long as you credit properly, you can.

Are you going to add X feature? Or I wanted to suggest Y Idea

If you want to see any feature added to the mod you can suggest it on the server and I will be more than open to brainstorm upon the idea, keep in mind it may end up a bit different or not added at all depending on the context.

When is minecraft version X is going to supported?

With new versions probably soon. with old versions probably never. If you want to create/maintain this mod for certain version hit me up on Discord.

Can i use X mod with this?

Very likely not, this however is not intended. if you find any issues be sure to report them on the github repo or Discord.
