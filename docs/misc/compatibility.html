BetterCombat Compatibility

epxzzysabers now includes full BetterCombat compatibility! The mod was previously tagged incorrectly by BetterCombat—this has been fixed in v0.0.4.

An additional extraneous compatibility datapack is available for enhanced integration. See GitHub releases for the link to the datapack.

Data-Driven Configuration

A new data-driven config system has been implemented, allowing weapon attributes to be customized through configuration files. Live reload is supported for quick testing and iteration during development.

See the mod's GitHub documentation for config file structure and examples.

Shader Support

Saber renderers have been updated to support external shaders for enhanced visual effects. The Rotary Saber entity now also supports shader integration for more dynamic visuals.

Shader support has been tested with the Oculus mod and popular shaderpacks including: Photon, Complementary Unbound, Complementary Reimagined, Rethinking Voxels (Beta), and BSL.
