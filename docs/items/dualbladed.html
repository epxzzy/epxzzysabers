Usage

Activating

To activate the item press [SHIFT + RMB]. Same can be done to turn it off. If a player throws (or fatfingers [Q]) an active saber from their mainhand it automatically turns off.

Attacking

When the item is active, it does 16hp of damage when left-clicked on an entity (or player). However like any other saber item, this can be countered via Blocking. A left-click can also deflect any incoming(in parry range, or in 4 block radius for this specific item) projectiles (given that the projectile is in a somewhat linear path, unlike shulkers for example), but timing would have to be precise.

Blocking

Dual Bladed Saber, like any other saber item, when right-clicked, will block any damage coming from a saber item, and even projectiles if timed properly. Can be held down for about ~1.75 seconds which if exceeded will cause the projectile to do damage.
