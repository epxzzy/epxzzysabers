Usage

Single Bladed Saber has two functions: 1. changing colour, 2. assigning stances. Right-Clicking on a Single Bladed Saber should bring up a Tinting menu by default. To change menus there are tabs button on the left side. NOTE: tab buttons may or may not change inputting an item, special items will have access to special tabs.

Tinting Menu

Single Bladed Saber can change the colour of the given saber item. By Default 3 HSL sliders are given. These can be toggled to RGB sliders upon pressing [SPACE]. if you press [PERIOD] an animated RGB colour is toggled, this however does not override the previous colour, meaning when pressed [PERIOD] again, the original colour of the item will be assigned.

Stance Menu

Single Bladed Saber can change the stance performed via the saber item. {TODO: LINK STANCING HERE}. There are Seven stance options provided, simply pick one and it will be given to the item. NOTE: every saber item has a unique stance, if you wish to change stances on the go, that is not possible UNLESS you have several saber items with different stances assigned. A preview of the stance is given before you assign it to the item, making it easier to choose one.
