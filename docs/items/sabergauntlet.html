Usage

Activating

To activate the item press [SHIFT + RMB]. Same can be done to turn it off. If a player throws (or fatfingers [Q]) an active saber from their mainhand it automatically turns off.

Attacking

When the item is active, it does 12hp of damage when left-clicked on an entity (or player). However like any other saber item, this can be countered via Blocking. A left-click can also deflect any incoming(in parry range, or in 3 block radius for this specific item) projectiles (given that the projectile is in a somewhat linear path, unlike shulkers for example), but timing would have to be precise.

Blocking

Saber Gauntlet, like any other saber item, when right-clicked, will block any damage coming from a saber item, and even projectiles if timed properly. Can be held down for about ~1.75 seconds which if exceeded will cause the projectile to do damage.

Disruption

Pressing [SABER-ABILITY] key lets out a superquick electromagneticalphaquazarradiation wave that causes nearby saber items to missbehave, either turning them off, turning them off with cooldown, or if a player is flying just cause them to straight up fall with both item turned inactive WITH cooldown. this happens to every entity holding any type of saber item in a SPHERICAL radius of 16 blocks. this however can be countered via riptide or SaberFlight.

SuperCharge

Holding down right-click for 40 ticks (~ 2 seconds) on an active saber gauntlet will activate supercharge. While the saber is supercharged player is given +3 block reach and 20hp of attack damage while held in mainhand. However supercharge lasts for 160 ticks (~ 8 seconds). This ability does not have a cooldown nor will it stack if activated in continuous manner.
