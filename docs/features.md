# Features

- Multiple saber types (single, dual, rotary, crossguard, pike, gauntlet, hybrid blaster)
- Parry and deflection mechanics (swing-based projectile redirection)
- Blocking with sabers (right-click block behavior)
- Stances and posing via Kyber Station
- Saber throw and projectile attacks (plasma bolts, rotary blades)
- Custom renderers and player pose animations
- Creative tabs and kyber rarity items
